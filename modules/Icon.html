# Icon

在项目框架中，图标的使用方式分为三种

  • 使用本地 Svg 文件作为图标
  • 使用 Element 提供的 Icon
  • 使用 阿里巴巴矢量库open in new window

在如上前提下，对 icon 进行了组件封装，通过统一的组件使用 icon

# 使用本地 Svg 图标

在项目框架下， src/assets/icons/svg 目录用来存放本地 Svg

组件 icon 传递的是 Svg 资源的名称

<template>
    <div class="icon-wrapper">
        <g-icon icon="caiwuriqi" />
        <g-icon icon="home-card-warning-icon" />
        <g-icon icon="dongtaikouling" />
        <GIcon icon="org" />
    </div>
</template>

<style lang="scss" scoped>
.icon-wrapper {
    svg {
        font-size: 24px;
        margin-right: 10px;

        &:last-of-type {
            color: #409eff !important;
        }
    }
}
</style>

# 使用 Element 提供的 icon

组件 icon 传递的是 Element 提供的 icon 的类名

请参考 Element Iconopen in new window

<template>
    <div class="icon-wrapper">
        <g-icon icon="el-icon-eleme" />
        <g-icon icon="el-icon-goods" />
        <g-icon icon="el-icon-success" />
    </div>
</template>

<style lang="scss" scoped>
.icon-wrapper {
    i {
        font-size: 24px;
        margin-right: 10px;

        &:last-of-type {
            color: #409eff;
        }
    }
}
</style>

# 使用 阿里巴巴矢量库 提供的 icon

注意

在 IconFontopen in new window 官网中存在一个图标项目，如需添加图标，请注册账号并联系张云超，邀请加入项目后自行添加图标 （现有图标预览open in new window）

组件 icon 传递的是下载到本地的图标资源的 Font Class

<template>
    <div class="icon-wrapper">
        <g-icon icon="ali-icon-wushuju" />
        <g-icon icon="ali-icon-xiangzidakai" />
        <g-icon icon="ali-icon-applications" />
    </div>
</template>

<style lang="scss" scoped>
.icon-wrapper {
    i {
        font-size: 24px;
        margin-right: 10px;

        &:last-of-type {
            color: #409eff;
        }
    }
}
</style>
