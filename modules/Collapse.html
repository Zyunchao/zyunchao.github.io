# Collapse

在 Element Collapseopen in new window 的基础上进行二次封装的折叠面板

  • 定制样式
  • 便捷传递表单、表格的配置
  • 灵活自定义内容

# 便捷传递表单、表格

组件内置了表单、表格，只需要传递对应的配置，即可生成对应的组件

# 自定义内容

和 el-collapse-item 一直，传递默认 slot 即可

# CollapseItem Attributes

参数说明类型默认值
formConfig表单配置FormPropsnull
tableConfig表格配置TableConfignull
height高度，适用于表格number300
